# my-visualization-app
Cung cấp giải pháp trực quan hóa dữ liệu tiện dụng, miễn phí, và không yêu cầu cài đặt phần mềm.   Áp dụng các nguyên tắc Kỹ thuật Phần mềm để tạo ra một sản phẩm ổn định, dễ sử dụng, và có khả năng mở rộng.   Chứng minh tính khả thi của việc kết hợp các Framework mạnh mẽ (Python Dash/Plotly) để giải quyết các vấn đề thực tế.
